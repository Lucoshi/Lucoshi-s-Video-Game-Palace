The Rayman O'Meter is used to calculate the distance covered with your mouse (it only works on Windows)! We'll leave it up to you to find out your performance: you're in for a surprise!

Once you have covered certain distances, you can receive prizes!

Once installed, the software application is launched each time you start your computer. It keeps the distance covered by your mouse in the memory.
To install it:

1 - Download the application
2 - Run the file to install the Rayman O'Meter
3 - That's it!!!
